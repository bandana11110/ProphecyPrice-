# 🏠🔮 ProphecyPriceAI

🚀 My First Machine Learning Project

ProphecyPriceAI is a machine learning-powered web application built using Python, Streamlit, and Scikit-Learn. The application predicts Boston house prices based on key housing features and provides instant predictions through an interactive and user-friendly interface.

This project marks my first step into the world of Machine Learning, where I learned data preprocessing, model training, prediction, deployment, and web application development.

🌐 Live Demo: https://huggingface.co/spaces/Bandana17/ProphecyPrice-AI

## 🚀 How It Works

1. Enter housing-related parameters.
2. The trained machine learning model processes the input data.
3. The model predicts the estimated house price.
4. Results are displayed instantly through the Streamlit interface.
